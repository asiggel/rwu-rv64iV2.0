The \asimemg peripheral is the internal instruction memory of the \aschip.

\paragraph{General Features: }
\begin{itemize}
  \item 32\;kByte ROM (8192 words/instructions)
  \item Linear addressable
  \item 32\;bit data width
  \item Wishbone bus slave
\end{itemize}

\paragraph{Synchronous Mode: }
\begin{itemize}
  \item 32 data bits (write; program download via JTAG)
\end{itemize}

\paragraph{Asynchronous Mode: }
\begin{itemize}
  \item 32 data bits (read; normal operation)
\end{itemize}
